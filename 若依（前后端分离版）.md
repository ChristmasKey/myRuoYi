## 一、什么是若依？

开源项目，学习开源项目的目标：

1、学习如何使用，减少自己的工作量

2、学习优秀开源项目底层的编程思想，设计思路，提高自己的编程能力

官网：http://www.ruoyi.vip/

基础：Spring Boot + Vue

环境要求：

- JDK1.8
- MySQL
- Redis
- Maven
- Vue
